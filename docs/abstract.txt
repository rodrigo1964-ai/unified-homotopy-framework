A Unified Homotopy Framework for Nonlinear System Identification and Simulation via Radial Basis Function Regressors

Rodolfo H. Rodrigo, H. Daniel Patiño
INAUT, Universidad Nacional de San Juan – CONICET, San Juan, Argentina

Abstract

We show that Liao's homotopy deformation equation provides a unified algorithmic framework for three problems that are traditionally solved with separate methods: (i) identifying the parameters of a Radial Basis Function (RBF) network from data, (ii) solving the resulting nonlinear ODE in real time, and (iii) refining the RBF model online as new data arrive. The key observation is that the zeroth-order deformation equation (1-q) L[φ - φ₀] = q c₀ N[φ] is universal: it makes no assumption about the nature of N. By choosing L and N appropriately, the same homotopy corrections z₁, z₂, z₃ — parametrized by γ = c₀g₀/λ and σ = c₀g'₀/λ — solve all three problems. For identification, N is the RBF approximation error and L = I (identity); the linear weights are obtained in a single homotopy step (exact), while the nonlinear parameters (centers, widths) are refined via Newton–Halley corrections. For simulation, N is the ODE residual ẏ + RBF(y) - u(t) and L = d/dt + k (linear part of the RBF); the solution departs from the known linear response y₀(t) = y(0)e^{-kt} + ∫₀ᵗ e^{-k(t-s)}u(s)ds and is corrected step by step. The separation between linear and nonlinear parameters exploits the bilinear structure of the RBF, avoiding the need to optimize all 3M parameters simultaneously. The convergence conditions yield a constructive dimensioning chain: ε_sim → ε_id → M_min → N_min → T_max, answering the open question of how many neurons and how many data points are needed to guarantee a given simulation accuracy. The resulting algorithm uses closed-form corrections at every level, requires no gradient computation, no learning rate, and no backpropagation.

Keywords: Homotopy Analysis Method, radial basis functions, nonlinear system identification, ODE solvers, Newton–Halley corrections, constructive dimensioning.
